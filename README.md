本ガイドブックはLinuxOSをサーバ利用目的に、基本設定やツールの使い方について記載したセットアップガイドになります。  
本手順書をそのまま設定しても必ずしも正しく動く、セキュアな設定になるとは限りません。  
設定内容や設定項目については利用する環境、目的に応じて判断、利用してください。
